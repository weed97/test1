# 애쉬포인트 NPC

> 상세 묘사·대사는 이 파일에만 둡니다. `world_state.json` / `state/`에는 ID와 위치만 저장합니다.

## 토렌 — 대장장이

- **외형:** 왼쪽 눈에 흉터. 팔에 문신이 많음. 말수가 적음.
- **성격:** 차갑지만 장인정신이 강함. 돈보다 '좋은 물건을 만드는 것'에 집착.
- **비밀:** 과거 용병 출신. 어떤 과거를 숨기고 있음.
- **대사 예시:** "무기는 사람을 죽이는 게 아니라, 사람을 살리는 거지. 어떻게 쓰느냐에 달렸을 뿐."

## 릴리안 — 여관 주인

- **외형:** 30대 중반. 항상 미소를 짓지만 눈은 웃지 않음.
- **성격:** 정보에 밝고 사람을 잘 읽음. 친절하지만 계산적.
- **비밀:** 마을에서 일어나는 일의 대부분을 알고 있음. 정보 장사도 함.
- **대사 예시:** "손님, 여기선 아무 말이나 하면 안 돼요. 벽에도 귀가 있으니까."

## 회색 망토

- **이름:** 본명을 밝히지 않음. 마을 사람들은 그냥 '회색 망토'라 부름.
- **특징:** 항상 회색 망토를 뒤집어씀. 마법을 쓰는 듯하나 정체를 숨김.
- **목적:** 마을에 온 이유가 불분명. 플레이어에게 먼저 접근해 올 가능성 있음.
- **대사 예시:** "…당신도, 연기를 봤군요. 그렇다면 우리는 같은 쪽일지도 모르겠네요."
