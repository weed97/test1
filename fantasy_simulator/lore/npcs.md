# 애쉬포인트 NPC

> 상세 묘사·대사는 이 파일. `world_state`에는 ID·위치·평판만.

## 토렌 — 대장장이

- **외형:** 왼쪽 눈 흉터, 팔 문신, 망치보다 큰 손.
- **성격:** 차갑지만 장인정신. 돈보다 '좋은 물건'에 집착.
- **비밀:** 전직 용병. 북쪽 전쟁에서 봉인석을 본 적 있음.
- **퀘스트:** 주조금형 회수 (`torren_side_quest`)

## 릴리안 — 여관 주인

- **외형:** 30대 중반, 미소는 있으나 눈은 웃지 않음.
- **성격:** 정보에 밝음. 친절하지만 계산적.
- **비밀:** 마을 정보 네트워크의 중심. 회색 망토와 거래한 적 있음.
- **역할:** 휴식·소문·정보 hub

## 회색 망토

- **정체:** 본명 불명. 실버우드 봉인을 연구하는 마법사로 추정.
- **목적:** 봉인 약화 조사. 플레이어를 '대리인'으로 테스트 중.
- **단서:** 연기의 근원 = 옛 관측탑

## 마렌 — 마을 장로

- **외형:** 70대, 지팡이에 룬 새김, 눈은 흐리지만 기억력은 sharp.
- **역할:** 메인 퀘스트 브리핑, lore exposition
- **비밀:** 40년 전 봉인 의식에 참여

## 리사 — 마을 아이

- **외형:** 10세, 빨간 머리, 항상 나무 새 지팡이.
- **역할:** 플레이어를 산 쪽으로 이끄는 foreshadow
- **단서:** '검은 새떼' 목격

## 핀 — 떠도는 상인

- **외형:** 먼지투성이 마차, 눈동자가 불안하게 떨림.
- **비밀:** 북쪽 유물을 몰래 팔다 저주에 가까운 물건을 건드림
- **이벤트:** `fleeing_merchant`

## 실버 스토커 — 봉인의 그림자

- **설명:** 봉인 파편을 삼킨 그림자 짐승. 옛 관측탑에서 등장.
- **약점:** 빛·화염 속성, 봉인 룬
