# 13 — 이세계 어포던스 (현실에서 못 하는 것)

## 원칙

**현실 동형(R3)** 은 몸·감각·사회 규범을 지키되, **판타지 계약** 으로 현실 불가능한 행위를 **명시적으로 허용**한다.  
플레이어는 「여기서만 가능한 것」을 체험하고, 그만큼 **세계에 기여·책임**을 갖는다 ([12](12_MICRO_TIME_AND_COCREATION.md)).

## 어포던스 맵

| 영역 | 현실 | 에르도리아 (엔진·설계) |
|------|------|------------------------|
| **마법** | 불가 | 스킬·원소·봉인 해제 — `RuleEngine.resolve_attack` + 주문 씨앗 |
| **사망** | 영구 | 로그아웃·Mnemosyne 리스폰·영혼 잔향 (설계 `02_ISEKAI_FRAME`) |
| **이동** | 물리 한계 | 텔레포트·비행·차원문 — travel moment 15분+ 연출 |
| **시간** | 선형 | Precision 시계·Story Beat 왜곡·휴식 점프 |
| **신체** | 고정 | 종족·각성·BCI 아바타 재구성 (`bci_meta`) |
| **기억** | 불완전 | Mnemosyne 샤드·NPC가 과거 대화 인용 |
| **사회** | 익명 한계 | 평판·세력·「마음이 들린다」 힌트 (미래 BCI) |
| **창조** | 소비만 | 월드빌딩 워크숍·승인 씨앗이 라이브 월드에 합류 |
| **경제** | 자본 제약 | 길드·봉인 재료·환상 화폐 루프 |
| **정치** | 권력 격차 | 6+α 세력에 **플레이어가 축이 됨** (`04_FACTIONS`) |

## 엔진에 연결하는 방법

| 어포던스 | 플래그 / 모듈 |
|----------|----------------|
| 마법 전투 | `combat`, spells in `characters/` |
| 분 단위 삶 | `temporal_mode=precision`, `minute_of_day` |
| 세계 기여 | `flags.world_building.*` |
| 육감 몰입 | `temporal_mode=nex|precision` + `[체감]` |
| 메인 운명 | `flags.main_story` phase gates |

## 내러티브 톤

- **유도:** 「당신이 남긴 기록이 비석에 새겨졌다」  
- **제한:** 메인 봉인·시즌 클라이맥스는 운영 스토리 — UGC는 **주변·서브** 우선  
- **안전:** R4 금지(현실 신원·의료 조작) — `10_REALITY_PARITY`

## 플레이 루프 예시

1. Precision으로 하루를 **분 단위**로 살다  
2. 숲에서 현실 불가능한 **그림자 길** 발견 (explore + 씨앗)  
3. 워크숍에 **소문 초안** 제출 → 승인 시 마을 NPC가 인용  
4. Phase 3에서 **자신이 쌓은 평판·기록**이 결말 가중치에 반영  

이 조합이 「이세계에서만 가능한 삶 + 내가 만든 세계」 경험의 핵심이다.
