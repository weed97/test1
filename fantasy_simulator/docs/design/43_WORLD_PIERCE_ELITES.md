# 43 — 세계 2~11위 방무 정예

설정: `config/world_apex_elites.json` · `config/damage_scaling.json` · `config/combatants.json`

---

## 방무 정예 (신화 10%)

| 항목 | 값 |
|------|-----|
| 세계 방무 인원 | **10명** (2~11위) |
| 신화템 고정뎀 | 등급별 base (신화 **100**) |
| **방무** | 본인 딜의 **10%** 만 관통 |
| 2위 방무 DPS | **1000**/s |
| 순위당 감소 | **-5%** (3위 950, … 11위 ~630) |
| **10명 합산** | ~**5000** 방무 DPS (실전 **3000~4000**) |
| 아서 100만 HP | **~200초** (합산 5000 기준) |

1위 아서 · 준신 방무는 **100%** (엑스칼리버).

## 스탯·딜 공식 (`damage_scaling.json`)

- STR 1당 **+0.88%**, INT **+0.92%**, 캐릭터 Lv 1당 **+0.1%**
- 무기: 일반1 · 고급2 · 희귀5 · 영웅10 · 전설50 · **신화100**
- 근거리 **×1.1**, 원거리 **×0.8**
- 스킬 계수: 마법 광역0.9 단일1.0 / 원거리 물리 0.7·0.8 / 근거리 물리 **0.6·1.1**
- 크리 **×1.5**

만렙 찍어도 맥스뎀 alone으로 아서 불가 — **방무 10% 정예 연합**이 정공.

## 리스크 — 아서 광역

정예가 **뭉치면** 아서 광역 **1초컷**. 산개·딜 타이밍·break가 필수.
