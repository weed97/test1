# 43 — 세계 2~11위 방무 정예

설정: `config/world_apex_elites.json` · `config/damage_scaling.json` · `config/combatants.json`

## 방무 정예 (신화 **3T** 무기만)

| 항목 | 값 |
|------|-----|
| 세계 방무 인원 | **10명** (2~11위) |
| 무기 | **신화 3T** (`mythic_3t_world_pierce_blade`) — 일반 신화100은 방무 없음 |
| **방무** | 본인 딜의 **10%** 만 관통 |
| 정예 HP | **500,000** (50만) |
| 2위 방무 DPS | **1000**/s |
| 순위당 감소 | **-5%** (3위 950, … 11위 ~630) |
| **10명 합산** | ~**5000** 방무 DPS (실전 **3000~4000**) |
| 아서 100만 HP | **~200초** (합산 5000 기준) |

1위 아서 · 준신 방무는 **100%** (엑스칼리버). 맥스뎀 플래그 대신 **STR 2000** + 엑스칼리버 공격 **500**.

---

## 아서 vs 정예 (DPS·광역)

| 항목 | 값 |
|------|-----|
| 방무 DPS | **100,000**/s |
| 타격 | 초당 **10**회 × **10,000** 방무/타 |
| 일반 광역 계수 | **×0.5** → 타당 **5,000** — **50만 HP 정예는 한 방에 안 죽음** |
| 성검 궁극기 | 쿨 **7200s**, HP **25%** 이하 위기 시 — **원샷** |
| 궁극기 반경 | **100px** — 밀집 시 **만 명** 즉사 가능, 회피 사실상 불가 |

정예가 뭉치면 궁극기·연속 광역에 취약. 산개·딜 타이밍·연합이 필수.

## 스탯·딜 공식 (`damage_scaling.json`)

- STR 1당 **+0.88%**, INT **+0.92%**, 캐릭터 Lv 1당 **+0.1%**
- 무기: 일반1 · 고급2 · 희귀5 · 영웅10 · 전설50 · **신화100** · 준신 엑스칼리버 **500**
- 근거리 **×1.1**, 원거리 **×0.8**
- 스킬 계수: 마법 광역0.9 단일1.0 / 원거리 물리 0.7·0.8 / 근거리 물리 **0.6·1.1**
- 크리 **×1.5**

만렙 찍어도 맥스뎀 alone으로 아서 불가 — **3T 방무 10% 정예 연합**이 정공.
