# 36 — 아이템 등급 · 레벨 우선 (준신~일반)

## 등급 (높은 순)

`준신` · `신화` · `전설` · `영웅` · `희귀` · `고급` · `일반`

설정: `config/item_grades.json`

---

## 황금 규칙: **등급 < 레벨**

| 대결 | 결과 |
|------|------|
| 준신검 + **무기 Lv1** | vs **일반검 + 무기 Lv999** → **무기 999 승** |
| 준신(결속) + **캐릭터 Lv1** | vs **일반 + 캐릭터 Lv999** → **캐릭터 999 승** |
| 신화 방어구 + 무기 Lv50 | vs 영웅 무기 Lv600 → **무기 레벨 쪽 우세** |

**아이템 등급**은 배율(약 **10%** 가중) — **캐릭터 Lv·무기 클래스 Lv** 가 체감을 결정.

```text
effective_power ≈ 0.42×char_lv + 0.48×weapon_class_lv + 0.10×grade_bonus
```

(구현 시 비선형·스킬·직업 Lv 추가)

---

## 등급별 착용·결속 문턱

| 등급 | 최소 캐릭터 Lv | 최소 무기 클래스 Lv |
|------|----------------|---------------------|
| 준신 | 800 | **999** |
| 신화 | 500 | 700 |
| 전설 | 200 | 400 |
| 영웅 | 80 | 150 |
| 희귀 | 30 | 60 |
| 고급 | 10 | 25 |
| 일반 | 1 | 1 |

- **준신** = 무기 **Lv999** 없으면 **결속 불가** (소지·운반만).
- **현재** 투핸드 999 + 엑스칼리버 결속 = **아서왕만** ([34](34_DEMIGOD_SOVEREIGN_EXCALIBUR.md)).

---

## 설계 의도

- “줄 좋은 템 줬다”로 끝나지 않음 — **무기 클래스를 키워야** 함.
- “줄 준신”도 **캐릭터·무기 레벨**이 낮으면 고레벨 일반 빌드에게 밀림.
- 엔드 **준신**은 **양 축 999** + 등급 — 극소수.
