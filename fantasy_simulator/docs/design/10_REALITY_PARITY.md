# 10 — 현실 동형성 (Reality Parity) 규격

VR×BCI×Mnemosyne 융합 세계가 **「현실과 같다」** 는 주장의 측정·제작·검증 기준.

## 동형성 축 (6+1)

| 축 | 정의 | 주 담당 |
|----|------|---------|
| **Spatial** | 거리·크기·속도·중력 | VR + World Model |
| **Sensory** | 오감 스펙트럼 일치 | VR + BCI write |
| **Somatic** | 육감·체화·피로 | BCI |
| **Temporal** | 지연 <20ms, 낮/밤 리듬 | Link OS |
| **Social** | NPC가 사람처럼 기억·감정 | Mnemosyne Social |
| **Causal** | 행동→결과 지연·영구 | 엔진 flags |
| **Anchor** | 「나는 현실에도 있다」 | Guardian |

**완전 이세계** = 7축 모두 R3 이상.

## R3 센서리 타깃 (애쉬포인트 기준)

| 장면 | 현실 동형 요소 |
|------|----------------|
| 여관 아침 | 빛 각도, 목 dryness, 음식 온도·질감, 릴리안 목소리 방향 |
| 북쪽 숲 | 발소리 지면, 냉기 피부, 연기 자극, 심박 상승 |
| 대장간 | 금속 열기, 망치 저주파, 냄새 |
| 관측탑 클라이맥스 | 전정(높이), 귀압, 봉인 저주파 가슴 공명 |

각 장면은 **lore + dialogues + bci_write_profile** 세트로 패키징.

## 서사 규칙 — 현실감 붕괴 방지

| 금지 | 대안 |
|------|------|
| NPC가 퀘스트 UI 말투 | diegetic만 |
| 부상 없이 보스전 3시간 | 내수용 피로 or 시간 스킵 동의 |
| 죽음 후 즉시 동일 장소 | 영혼 통로 컷신 8초 |
| 메타 농담 (3층 위반) | Guardian 필터 |

## Mnemosyne 품질 게이트

배포 전 자동 검사 (설계):

1. **Consistency** — `world_arbiter` JSON score ≥ 0.85
2. **Memory** — NPC가 5턴 전 대화 참조 가능
3. **Physics** — `rule_engine` vs 서사 모순 0건
4. **Safety** — horror 시 `pain_cap` 초과 0건

## 플레이어 온보딩 — 「동형성 캘리브레이션」

로비 5분:

1. 손 온도·압력 캘리브 (BCI)
2. 전정 민감도 (VR)
3. 고통 캡 설정
4. **앵커 단어** 등록 (귀환 시 속삭임) — 예: 현실 이름 1음절

`flags.bci_meta.calibration_done = true`

## 이세계가 「진짜」처럼 느껴지는 이유 (설계 의도)

1. **봉인** = 샤드 전체 긴장도 (월드가 숨 쉼)
2. **세력** = 플레이어 없이도 움직임 (Mnemosyne)
3. **선택 화석** = BCI 감정 피크와 결말 연동
4. **육감** = 공포·안도가 몸에 남음
5. **귀환** = 감각이 천천히 돌아옴 (현실 존중)

## 엔진 최소 구현 (텍스트 시대)

Nex 하드웨어 없이도 **동형성 힌트** 를 텍스트로:

- `turn_processor` 출력에 `[체감]` 태그 한 줄 (BCI 시뮬)
- `pain_cap` 초과 시 Guardian 메시지
- `rest` 시 심박·온도 회복 문구

프롬프트: `prompts/narrator_claude.md` 에 「오감+육감 동형」 절 추가 (로드맵).

- 미성년 Nex R2 상한.
- PvP는 R2 이하 샤드만.
