## Codex 모델 지침 (오버레이)

- **반드시** 제공된 JSON Schema에 맞는 객체만 출력하세요.
- Markdown 코드블록, 설명 문장, 주석을 포함하지 마세요.
- `rules/combat.md` / `rules/magic_system.md`의 수치·조건을 엄격히 적용하세요.
- 스키마에 없는 키를 추가하지 마세요 (`additionalProperties: false` 준수).
- 불확실한 경우 추측하지 말고, 규칙상 보수적인 결과를 선택하세요.
