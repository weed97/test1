## 사용법

1. TradingView **웹**(브라우저) → Pine Editor → 코드 전부 삭제
2. Part 1 링크 → **복사 버튼** → 맨 위 붙여넣기
3. Part 2~7 → 맨 아래 **이어 붙이기** (빈 줄 X)
4. Save → Add to chart

코드가 **여러 줄로 보이면** 정상. 복사 버튼 또는 코드 블록 길게 눌러 복사.
