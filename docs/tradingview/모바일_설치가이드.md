# 모바일 TradingView 설치 가이드

PC에서는 Raw 한 번에 되는데, **휴대폰 Raw 페이지는 코드가 한 줄로 보이고**, 복사해도 줄바꿈이 깨지는 경우가 많습니다.

→ **복사 버튼 페이지**를 쓰세요. (줄바꿈 유지)

---

## ✅ 권장: 복사 버튼 페이지 (모바일)

### 1) 아래 링크를 **Safari / Chrome**에서 열기

**한 페이지에 Part 1~6 전부:**

```
https://cdn.jsdelivr.net/gh/weed97/test1@cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/index.html
```

(안 되면 백업)

```
https://htmlpreview.github.io/?https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/index.html
```

### 2) TradingView **웹**(브라우저) → Pine Editor

- TradingView **앱** 말고 **브라우저**에서 tradingview.com 접속
- Pine Editor → 기존 코드 **전부 삭제**

### 3) Part 1 → 6 순서

1. 복사 페이지에서 **「Part 1 복사」** 초록 버튼 탭
2. Pine Editor **맨 위**에 붙여넣기
3. 에디터 **맨 아래**로 스크롤
4. **Part 2 복사** → 이어 붙이기
5. Part 3 ~ 6 반복
6. **Save** → **Add to chart**

### 4) 성공 확인

맨 위가 **두 줄**로 보여야 함:

```
//@version=6
// ═══...
```

`//@version=6indicator(` 처럼 한 줄이면 → Part 1부터 다시.

## 에러 시

| 증상 | 해결 |
|------|------|
| `Mismatched input` | Part 1~6 처음부터 다시 (복사 버튼 사용) |
| 복사 버튼 안 됨 | 코드 `<pre>` 영역 길게 눌러 직접 복사 (페이지 소스에 줄바꿈 있음) |
| `Script too heavy` | VP Lookback 150, VP Bins 24 |
