# 모바일 TradingView 설치 가이드

**Raw GitHub = 모바일에서 한 줄로 보임** → 복사 버튼 페이지만 쓰세요.

---

## ✅ 7분할 복사 페이지 (권장)

**Safari / Chrome**에서 열기:

```
https://cdn.jsdelivr.net/gh/weed97/test1@cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/index.html
```

### 순서

1. TradingView **웹**(앱 X) → Pine Editor
2. 기존 코드 **전부 삭제**
3. **Part 1** 복사 → **맨 위**에 붙여넣기
4. **Part 2 ~ 7** 복사 → **맨 아래에 이어 붙이기** (중간에 빈 줄 넣지 않기)
5. Save → Add to chart

### 7분할 구성 (섹션 단위, 끊김 최소)

| Part | 내용 | 줄 수(대략) |
|------|------|------------|
| 1 | `//@version=6` · 설정 · Session VWAP · σ 밴드 | ~132 |
| 2 | Weekly/Monthly VWAP · VWAP Zones | ~99 |
| 3 | Volume Profile · POC | ~161 |
| 4 | Buy/Sell Wall | ~125 |
| 5 | Support/Resistance (전체) | ~234 |
| 6 | Wall Pressure 테이블 | ~116 |
| 7 | Dashboard · 시그널 · Alerts | ~149 |

Part 1만 `//@version=6`가 있습니다. Part 2~7은 바로 이어 붙이면 됩니다.

### 성공 확인

맨 위:

```
//@version=6
// ═══...
```

한 줄로 `//@version=6indicator(` 붙어 있으면 → Part 1부터 다시.
