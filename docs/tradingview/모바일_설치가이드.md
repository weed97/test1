# 모바일 TradingView 설치 가이드

PC에서는 한 번에 붙여넣기가 되는데, **휴대폰에서는 줄바꿈·들여쓰기가 깨져** Pine Script가 한 덩어리로 붙는 경우가 많습니다.

아래 방법을 **위에서부터** 순서대로 시도하세요.

---

## 절대 하지 말 것

| 하면 안 되는 것 | 이유 |
|----------------|------|
| 카카오톡·텔레그램·디스코드에서 코드 복사 | 줄바꿈이 사라지거나 공백이 깨짐 |
| Cursor/채팅창에서 긴 코드 복사 | 모바일에서 한 줄로 합쳐짐 |
| TradingView **앱** 안에서 GitHub 링크 열기 | 복사 품질이 브라우저보다 나쁜 경우 많음 |

---

## 방법 1 — 모바일 브라우저 + Raw (가장 먼저 시도)

**Safari(아이폰) 또는 Chrome(안드로이드)** 사용. TradingView 앱 말고 **브라우저**로 접속합니다.

### 1) Raw 파일 열기

아래 링크를 **브라우저 주소창**에 직접 입력하거나 붙여넣기:

```
https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/Crypto_Precision_Engine.pine
```

### 2) 전체 선택 후 복사

- **iPhone Safari**: 화면 길게 누르기 → **모두 선택** → **복사**
- **Android Chrome**: 메뉴(⋮) → **모두 선택** → **복사**

> 팁: 화면이 작으면 **데스크톱 사이트 요청**(Chrome 메뉴) 후 다시 시도하면 선택이 쉬울 때가 있습니다.

### 3) TradingView 웹에서 붙여넣기

1. 브라우저에서 [tradingview.com](https://www.tradingview.com) 로그인
2. 차트 하단 **Pine Editor** 탭
3. 기존 코드 **전부 삭제**
4. **맨 위**에 커서 두고 붙여넣기
5. **Save** → **Add to chart**

### 4) 붙여넣기 후 확인 (중요)

에디터 **맨 위**가 이렇게 보여야 합니다:

```
//@version=6
// ══════════════════════════════════════
indicator(
```

한 줄에 `//@version=6indicator(` 처럼 **붙어 있으면 실패** → 방법 2로 가세요.

---

## 방법 2 — 6조각 나눠 붙이기 (모바일 전용, 권장)

코드를 **섹션 단위 6개 파일**로 나눠 두었습니다.  
**Part 1 → Part 6 순서**로, **이어 붙이기**만 하면 됩니다.

| 단계 | Raw 링크 (브라우저 주소창에 입력) |
|------|-----------------------------------|
| **1** | https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/chunks/part1.pine |
| **2** | https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/chunks/part2.pine |
| **3** | https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/chunks/part3.pine |
| **4** | https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/chunks/part4.pine |
| **5** | https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/chunks/part5.pine |
| **6** | https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/mobile/chunks/part6.pine |

### 붙이는 순서

1. Pine Editor 열기 → 기존 코드 **전부 삭제**
2. **Part 1** Raw 페이지 → 모두 선택 → 복사 → 에디터에 붙여넣기
3. 에디터에서 **맨 아래로 스크롤** → 줄 끝에 커서
4. **Part 2** 복사 → **바로 이어서** 붙여넣기 (중간에 빈 줄 넣지 않아도 됨)
5. Part 3 ~ 6도 **같은 방식** 반복
6. Save → Add to chart

### 각 Part가 맞게 붙었는지 확인

| Part | 맨 앞에 보이는 줄 (대략) |
|------|--------------------------|
| 1 | `//@version=6` … `SECTION 1: VWAP` |
| 2 | `SECTION 2: VOLUME PROFILE` |
| 3 | `SECTION 3: BUY WALL` |
| 4 | `SECTION 4: SUPPORT` |
| 5 | `SECTION 5: WALL PRESSURE` |
| 6 | `SECTION 6: MASTER DASHBOARD` … `alert(` |

Part 2~6는 `//@version=6`가 **없는 것이 정상**입니다 (Part 1에만 있음).

## 방법 3 — 메모 앱 경유 (줄바꿈 살리기)

Raw에서 복사해도 TradingView에 붙일 때 깨지면:

1. **iPhone 메모** / **Samsung 메모** 앱 열기
2. Raw 페이지에서 복사한 내용을 **메모에 먼저** 붙여넣기
3. 메모에서 줄이 여러 줄로 보이는지 확인
4. 메모에서 **다시 전체 선택 → 복사**
5. TradingView Pine Editor에 붙여넣기

---

## 방법 4 — PC 사용 (가장 확실)

- PC 브라우저에서 [Raw 전체 파일](https://raw.githubusercontent.com/weed97/test1/cursor/crypto-precision-engine-pine-94d6/docs/tradingview/Crypto_Precision_Engine.pine) 열기
- `Ctrl+A` → `Ctrl+C` → Pine Editor `Ctrl+V`
- 저장하면 **같은 TradingView 계정**으로 모바일에서도 스크립트가 동기화됩니다

---

## 컴파일 에러가 날 때

| 증상 | 해결 |
|------|------|
| `Mismatched input` / `Undeclared identifier` | 들여쓰기 깨짐 → Part 1~6 **처음부터 다시** |
| 코드가 한 줄로 보임 | 방법 2(6조각) 또는 PC 사용 |
| `Script too heavy` | 설정에서 VP Lookback 150, VP Bins 24로 줄이기 |

자세한 PC 설치: [TRADINGVIEW_설치.md](./TRADINGVIEW_설치.md)
