# 무료 오더북 가이드 — Binance / Bybit (오픈소스·API 키 불필요)

## 한 줄 답

**네. Binance·Bybit 공개 오더북(Market Data)은 API 키 없이 무료입니다.**

CoinGlass처럼 유료 구독이 아니라, 거래소가 공개하는 **공식 무료 API**입니다.

---

## 무료 vs 유료 구분

| 구분 | API 키 | 비용 | 예시 |
|------|--------|------|------|
| **오더북·시세·캔들** (공개) | ❌ 불필요 | **무료** | depth, ticker, klines |
| **주문·잔고·출금** (비공개) | ✅ 필요 | 무료 (거래 수수료만) | place order, balance |

오더북만 보려면 **회원가입·API 키·돈 전혀 필요 없음**.

---

## Binance 무료 엔드포인트

| 시장 | REST | WebSocket |
|------|------|-----------|
| Spot | `GET https://api.binance.com/api/v3/depth?symbol=BTCUSDT&limit=20` | `wss://stream.binance.com/ws/btcusdt@depth20@100ms` |
| USDT-M 선물 | `GET https://fapi.binance.com/fapi/v1/depth?symbol=BTCUSDT&limit=20` | `wss://fstream.binance.com/ws/btcusdt@depth20@100ms` |

문서: https://developers.binance.com/docs/binance-spot-api-docs/rest-api#order-book

---

## Bybit 무료 엔드포인트

| 시장 | REST | WebSocket |
|------|------|-----------|
| Linear (USDT Perp) | `GET https://api.bybit.com/v5/market/orderbook?category=linear&symbol=BTCUSDT` | `wss://stream.bybit.com/v5/public/linear` |
| Spot | `category=spot` | `wss://stream.bybit.com/v5/public/spot` |

문서: https://bybit-exchange.github.io/docs/v5/market/orderbook

---

## 레포 스크립트 (pip·API 키 불필요, stdlib만)

```bash
# Binance USDT-M 선물 (TradingView BINANCE:BTCUSDT.P 와 동일 계열)
python3 scripts/tradingview_orderbook_feed.py --exchange binance --market futures --ws

# Bybit ETH 선물 — WebSocket 실시간
python3 scripts/tradingview_orderbook_feed.py --exchange bybit --symbol ETHUSDT --market futures --ws

# REST 1초 폴링 (방화벽 때문에 WS 안 될 때)
python3 scripts/tradingview_orderbook_feed.py --exchange binance --market futures --interval 1
```

## 오픈소스 프로젝트 (같은 무료 API 사용)

| 프로젝트 | 언어 | 용도 |
|----------|------|------|
| [ccxt](https://github.com/ccxt/ccxt) | JS/Python | 100+ 거래소 통합, `fetch_order_book()` |
| [python-binance](https://github.com/sammchardy/python-binance) | Python | Binance 전용 |
| [pybit](https://github.com/bybit-exchange/pybit) | Python | Bybit 공식 SDK |
| [hummingbot](https://github.com/hummingbot/hummingbot) | Python | MM 봇 + 오더북 |
| [cryptofeed](https://github.com/bmoscon/cryptofeed) | Python | 실시간 피드 통합 |

이들도 **공개 market data는 키 없이** 동작합니다.

## CoinGlass vs Binance/Bybit

| | CoinGlass | Binance / Bybit 직접 |
|--|-----------|----------------------|
| 비용 | API 유료 ($29/월~) | **무료** |
| 데이터 | 여러 거래소 합산·대형벽 | 해당 거래소 **실제 호가** |
| 오픈소스 | ❌ | API 문서 공개 + OSS 클라이언트 다수 |
| Pine 연동 | ❌ | ❌ (터미널 병행) |

TradingView에서 `BINANCE:BTCUSDT` 차트 보면서, 터미널에서 **같은 거래소 무료 depth** 보면 됩니다.

## Pine Script와의 관계

```
Binance/Bybit 무료 API  →  Python 스크립트  →  터미널 실시간 오더북  ✅
Binance/Bybit 무료 API  →  Pine Script 직접  ❌ (HTTP 불가)
TradingView 캔들 피드   →  Pine 인디케이터   ✅ (이미 실시간)
```

**정리:** 오더북은 Binance/Bybit 무료 API + 오픈소스 스크립트로 충분합니다. CoinGlass API는 필수가 아닙니다.
