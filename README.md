# 중세 판타지 시뮬레이터

브라우저에서 바로 실행할 수 있는 중세 판타지 왕국 경영 시뮬레이터입니다.
플레이어는 은빛가시 왕국의 영주가 되어 자원, 건물, 병력, 모험가, 위협도를 관리합니다.

## 주요 기능

- 계절 단위 턴 진행과 자원 생산/소모
- 농장, 벌목소, 마법사의 탑, 병영, 주점, 성벽 건설
- 모험가 의뢰, 결계 의식, 축제, 징집 등 선택형 명령
- 랜덤 사건과 승패 조건
- 브라우저 `localStorage` 기반 자동 저장

## 실행

의존성 설치 없이 `index.html`을 브라우저로 열면 플레이할 수 있습니다.

```bash
python3 -m http.server 8000
```

그 다음 `http://localhost:8000`에 접속하세요.
